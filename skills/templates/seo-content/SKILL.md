# SEO & Content Optimization

## Overview

This skill provides comprehensive SEO and content optimization guidelines for creating high-ranking, AI-visible content across web pages, blog posts, and marketing materials.

## Core Principles

1. **Search Intent First** — Match content to what users actually search for
2. **E-E-A-T Signals** — Experience, Expertise, Authoritativeness, Trustworthiness
3. **GEO/AEO Ready** — Optimize for AI engines (ChatGPT, Perplexity, Google AI Overviews)
4. **Semantic Structure** — Use proper heading hierarchy and schema markup
5. **Mobile-First** — Responsive, fast, accessible

## On-Page SEO Checklist

- Title tag: 50-60 chars, primary keyword near start
- Meta description: 150-160 chars, compelling CTA
- H1: one per page, matches search intent
- H2-H6: logical hierarchy, include LSI keywords
- Internal links: 3-5 per 1000 words
- Image alt text: descriptive, keyword-aware
- URL: short, hyphenated, keyword-rich

## Content Structure

### Blog Posts / Articles
```
H1: Primary keyword + hook
  H2: What is [topic]?
  H2: Why [topic] matters
  H2: How to [action] (step-by-step)
    H3: Step 1...
    H3: Step 2...
  H2: Best practices / Tips
  H2: FAQ (use FAQPage schema)
  H2: Conclusion + CTA
```

### Landing Pages
```
Hero: Headline + subline + CTA
Social proof: logos, stats, testimonials
Features: benefit-driven, not feature-driven
FAQ: address objections
Final CTA: urgency + value prop
```

## AI Visibility (GEO)

- Structure content with clear Q&A patterns
- Use definition lists and concise answers
- Include statistics with sources
- Add `llms.txt` for AI crawler guidance
- Implement FAQPage and HowTo schema markup
